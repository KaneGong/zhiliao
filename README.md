# 知料 ZhiLiao — 食品原料智能平台
> 
> 🌐 域名：zhiliao-ai.cn

> 开放的食品原料信息与推荐平台，连接上游原料供应商与下游食品企业
> 
> Logo 概念：以「知了」（cicada）为形象，谐音「知料」——知道原料，洞察原料。

## 定位

**知料（ZhiLiao）—— 不是 ANG 的内部工具，而是一个开放的原料智能平台。**

- 第一阶段：以 ANG（爱联康营养）的产品数据为核心种子数据
- 第二阶段：开放给其他原料供应商接入
- 最终目标：打通食品原料上下游的信息壁垒

## 为什么是开放平台

1. **信息不对称是行业痛点** — 食品企业找原料靠人脉和展会，效率极低
2. **供应商需要曝光渠道** — 中小供应商缺乏数字化推广能力
3. **数据可以不断丰富** — 单一供应商的数据有限，多供应商数据才有比较价值
4. **Kane 在 ANG 的优势** — 能拿到一手资料，但不应局限于 ANG

## 核心功能

1. **原料智能推荐** — 根据产品需求推荐最佳原料组合
2. **法规合规检查** — 自动检查配方是否符合 GB 标准
3. **多供应商比价** — 同类原料跨供应商价格/规格对比
4. **保健食品/特医/婴配** — 扩展到特殊食品领域
5. **供应商入驻** — 供应商可上传自家产品数据

## 数据架构

```
suppliers/          ← 按供应商组织
  ang/              ← ANG（爱联康营养）—— 第一个种子供应商
    prochin/        ← Prochin 产品线（乳蛋白）
    novosana/       ← Novosana 产品线（鱼油/藻油）
  {future-supplier}/ ← 未来其他供应商
```

## 价格说明

- ANG 价格来自内部价目表，不代表最终成交价
