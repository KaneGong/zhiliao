## 1. 产品定位

**一句话：帮食品企业快速找到合适的原料，同时确保符合法规。**

知料是一个开放的食品原料智能平台，连接上游原料供应商与下游食品企业：
- **上游**：供应商展示产品数据（功能、规格、价格）
- **下游**：食品企业搜索原料、获取 AI 推荐、检查法规合规

### MVP 阶段范围
- 仅接入 ANG（爱联康营养）作为种子供应商
- 60+ 产品数据已就绪（Prochin 乳蛋白 + Novosana 鱼油/藻油）
- 验证核心假设：食品企业研发/采购人员是否愿意用 AI 推荐找原料

---

## 2. 目标用户

### 主要用户：食品企业研发/采购人员
- **角色**：研发工程师、产品经理、采购专员
- **场景**：开发新产品时需要找合适的原料供应商
- **痛点**：
  - 找原料靠人脉和展会，效率低
  - 不确定原料是否符合 GB 法规
  - 同类原料跨供应商比价困难
  - 信息分散，缺乏一站式平台

### 次要用户：原料供应商销售/市场人员
- **MVP 阶段**：由 Kane 手动录入数据，不需要供应商后台
- **后期**：供应商自主上传和更新产品数据

## 3. 核心功能（MVP）

### 3.1 原料搜索
**用户故事：** 作为食品研发人员，我想按品类/功能搜索原料，快速找到候选产品。

**功能描述：**
- 按品类筛选：乳蛋白、鱼油、藻油、植物蛋白、功能性原料等
- 按功能筛选：免疫调节、体重管理、骨骼健康、抗氧化等
- 按供应商筛选：当前仅 ANG
- 关键词搜索：支持中英文产品名、功能描述搜索
- 搜索结果列表：产品名 + 品类 + 供应商 + 一句话功能描述

### 3.2 AI 智能推荐
**用户故事：** 作为食品研发人员，我输入产品需求描述，AI 帮我推荐最合适的原料组合。

**功能描述：**
- 输入框：自由文本描述需求（如"我要做一款高蛋白低糖的运动后恢复饮料"）
- AI 输出：
  - 3-5 个推荐原料方案
  - 每个方案包含：原料名称 + 功能 + 建议添加量 + 法规状态 + 参考价格区间
  - 每个推荐标注数据来源（如"来源：Glanbia Prolibra 产品规格书"）
- 技术实现：RAG（检索增强生成）+ MiMo API

**幻觉控制机制：**
- AI 只能推荐数据库中已有的原料
- 每个推荐必须引用来源
- 法规检查用规则引擎，不靠 AI 判断
- 数据库没有的就说"暂无数据，建议向供应商确认"

### 3.3 法规速查
**用户故事：** 作为食品研发人员，我想快速查某个原料是否符合中国食品安全法规。

**功能描述：**
- 输入原料名或功能声称
- 系统检查：
  - GB 2760（食品添加剂使用标准）
  - GB 14880（食品营养强化剂使用标准）
  - 保健食品原料目录
- 输出：合规状态 + 具体条款引用 + 注意事项

### 3.4 供应商信息展示
**用户故事：** 作为采购人员，我想看到供应商的联系方式，方便进一步询价。

**功能描述：**
- 供应商详情页：公司简介 + 产品线概览 + 联系方式
- 产品详情页：完整规格 + 临床证据 + 法规状态 + 供应商信息
- 当前阶段：联系方式指向 ANG 官方渠道

## 6. 技术方案（概要）

### 前端
- **框架**：Next.js 14+（React）
- **部署**：Vercel（免费额度，MVP 够用）
- **UI 库**：Tailwind CSS + shadcn/ui

### 后端
- **框架**：Next.js API Routes 或独立 Node.js 服务
- **数据库**：PostgreSQL（Supabase 托管，免费 500MB）
- **向量搜索**：pgvector（语义搜索原料）

### AI 层
- **主模型**：MiMo V2.5（推荐生成）
- **搜索**：向量 Embedding + 语义搜索
- **法规**：规则引擎（GB 标准校验）

### 域名 & 部署
- **域名**：zhiliao-ai.cn（已注册）
- **服务器**：MVP 阶段用 Vercel（海外）+ Supabase（海外）
- **ICP 备案**：已提交，待审核

## 10. 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| AI 幻觉 | 推荐不准确，损害信任 | RAG + 规则引擎 + 引用来源 |
| 数据不全 | 推荐范围有限 | 先做 60 个核心产品，逐步扩充 |
| 用户不来 | 验证失败 | 利用 Kane 的行业人脉推广 |
| 竞品出现 | 市场被抢 | 先发优势 + 数据壁垒 |
| 法规风险 | 合规问题 | 法规检查用规则引擎，不做法律建议 |
